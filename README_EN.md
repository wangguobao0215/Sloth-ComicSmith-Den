## Feature Overview

1. **Script Parsing (with Dramatic Structure)**: Auto-identify three-act structure, inciting incident, midpoint, climax, and all-is-lost moment
2. **Character Consistency Design**: Four-view reference images, expression sheets, character arc tracking, outfit change logging
3. **Color Script**: Full-story color narrative blueprint with emotion-to-color auto-mapping
4. **Intelligent Storyboarding**: Professional shot grammar (framing + function), precise camera motion, 180-degree axis tracking, audio direction
5. **AI Image Generation**: Multi-model support, seed locking strategy, batch fault tolerance
6. **Video Synthesis**: Precise motion descriptions, transitions, subtitle burning
7. **Comic Layout**: HTML/CSS or Pillow composition for full comic pages

## Quick Start

1. Clone the repo
2. Initialize a project:
   ```bash
   cd Sloth-ComicSmith-Den/scripts
   python3 init_project.py my_comic_project
   ```
3. Follow the five-stage workflow in SKILL.md
4. Validate project files:
   ```bash
   python3 validate_project.py my_comic_project
   ```
5. Auto-generate color script:
   ```bash
   python3 generate_color_script.py my_comic_project
   ```
